# syncline-book-store
## To  start the server (book store app) run (from the root directory):

```bash
npm run server_start
```

Architecture:
 
 
 Login/register
  + Vue
  + Vuex 
  + Vue-router
  + Build: vite
  
 User App:
  + React
  + React Router
  + Redux
  + Build: webpack babel
  
 Server:
  + Express
  + Database: Mongo db
