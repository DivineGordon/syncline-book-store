# syncline-book-store
## To  start the server (book store app) run (from the root directory):

```bash
npm run server_start
```

## View page at http:localhost:8080/static/index.html

## To create admin user
```bash
npm run create_admin
```
Architecture:
 
 
 Login/register
  + Vue
  + Vuex 
  + Vue-router
  + Build: vite
  
 User App:
  + React
  + React Router
  + Redux
  + Build: webpack babel
  
 Server:
  + Express
  + Database: Mongo db
  
 Admin App:
 + React
 + React router
 + Redux
